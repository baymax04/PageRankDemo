PageRank是矩阵G 的特征向量q=Gq
	其中G=aS+(1-a)/n U
	S：根据网页的指向得到的原始矩阵
	n:是网页的数量
	U：全为1 的n阶矩阵
		0   0    0   0
	s=  1/3 0    0   1
		1/3 1/2  0   0
		1/3 1/2  1   0

	1.初始alpha 值 和阈值
	2.初始向量q
	3.打印向量q
	4.初始矩阵G
		1.计算获得初始矩阵
			1.初始化S矩阵
			2.初始化U矩阵
			3.计算一个数乘以矩阵
				aS
				(1-a)/n U
			4.计算两个矩阵之和
		
	5.打印矩阵G
	6.计算PageRank
		1.计算一个矩阵乘以一个向量
		2.计算两个向量之间的距离
	7.输出PageRank
